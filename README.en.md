# Adventure3d

**Adventure3D** is a simple 3D adventure game prototype developed using
the Unity Engine.

The project demonstrates core gameplay mechanics commonly found in
adventure games, including:

-   Character movement (run and jump)
-   Collection system (coins and items)
-   Object interaction
-   Destructible objects
-   Shooting mechanics
-   Enemy elimination system

This project was developed as a learning experience, focusing on 3D
gameplay systems and fundamental game development concepts in Unity,
during classes at the **EBAC** platform.

## Technologies and Tools Used

-   Game engine: Unity
-   Language: C#
-   Git for version control

## Controls

Move: **WASD**\
Jump: **Spacebar**\
Shoot: **X**\
Interact: **Z**
