# Adventure3d

**Adventure3D** é um protótipo simples de jogo de aventura em 3D
desenvolvido utilizando a Unity Engine.

O projeto demonstra mecânicas centrais de gameplay comumente encontradas
em jogos de aventura, incluindo:

-   Movimentação do personagem (correr e pular)
-   Sistema de coleta (moedas e itens)
-   Interação com objetos
-   Objetos destrutíveis
-   Mecânica de tiro
-   Sistema de eliminação de inimigos

Este projeto foi desenvolvido como experiência de aprendizado, com foco
em sistemas de gameplay 3D e conceitos fundamentais de desenvolvimento
de jogos na Unity, durante as aulas da plataforma **EBAC**.

## Tecnologias e Ferramentas Utilizadas

-   Motor de jogo: Unity
-   Linguagem: C#
-   Git para controle de versão
